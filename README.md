# Artale Boss Chaser (Pro) - 獵人儀表板

這是一款專為《Artale》玩家設計的高效率 Boss 擊殺紀錄工具。它能幫助你精確追蹤各個頻道的 Boss 重生時間，並透過「獵殺專注模式」讓你輕鬆在數千個頻道中切換，紀錄每一份戰利品。

---

## 🌟 核心強大功能

### 1. 🎯 獵殺專注模式 (Hunting Focus Mode)
當你選定一個 Boss 作為目標時，介面會切換為專屬的操作面板：
- **超大號頻道輸入：** 專為高頻率操作設計，一眼看清目前紀錄頻道。
- **自動跳頻系統：** 開啟後，每次紀錄完畢頻道會自動 +1，省去手動修改時間。
- **快速鍵支援：** 在頻道輸入框按下 `Enter` 鍵即可立即紀錄，配合自動跳頻，實現「秒速紀錄」。
- **自動全選：** 紀錄後輸入框會自動全選，讓你隨時可以手動微調頻道而不需先刪除舊數字。

### 2. 💎 精確掉寶與幸運頻道紀錄
不再只是記錄「有掉寶」，現在你可以分類標記：
- **🛡️ 裝備 (Equip)：** 記錄掉落的防具與武器。
- **📜 卷軸 (Scroll)：** 記錄掉落的各式捲軸。
- **🌟 星星 (Star)：** 標記為「幸運頻道」！當你打到大寶物時勾選此項，歷史紀錄中會出現發光的星星圖示，幫你找出最容易出貨的頻道。

### 3. ⏳ 智慧重生預估
- **動態倒數：** 卡片會即時顯示「最早」與「最晚」重生倒數。
- **狀態燈號：** 
  - 🟢 **可擊殺：** Boss 已過重生點，隨時可能出現。
  - 🟡 **即將重生：** 進入 2 分鐘預警期（開啟音效後會有提示音）。
  - 🔴 **冷卻中：** Boss 剛被擊殺，正在重生冷卻期。
- **進度條顯示：** 視覺化呈現重生的百分比。

### 4. 📱 完美 RWD 響應式支援
- 無論是電腦大螢幕還是手機，介面都會自動調整佈局。
- 手機版會將操作區固定在頂部，方便單手快速紀錄。

---

## 🚀 快速上手教學

1.  **選擇目標：** 點擊左側列表中的任何一個 Boss 卡片，右側會開啟「獵殺專注面板」。
2.  **設定起始：** 輸入你目前所在的頻道數值。
3.  **開始獵殺：** 
    - 擊殺 Boss 後，視情況勾選 **裝備 / 卷軸 / 星星**。
    - 直接按下面板上的 **「立即紀錄擊殺」** 或在輸入框按 **Enter**。
4.  **連續追蹤：** 系統會自動紀錄時間、頻道與掉寶，並自動跳到下一個頻道。
5.  **查看歷史：** 面板下方會顯示該 Boss 在各個頻道的最新狀態，標有 🌟 的就是你的幸運頻道！

## 💾 資料與隱私說明

- **本機存儲：** 所有的擊殺紀錄都只儲存在你的瀏覽器（LocalStorage）中。
- **完全隱私：** 你的資料不會上傳到任何伺服器，除非你主動進行備份分享。
- **備份建議：** 建議定期將資料匯出（目前可透過開發者工具），以防瀏覽器快取被清理導致紀錄遺失。

---

## 🛠️ 常見問題

- **頻道支援上限：** 系統支援 1 至 3000 個頻道。
- **資料清除：** 點擊「清空所有紀錄」會移除所有 Boss 的歷史，請謹慎使用。
- **音效提示：** 點擊右上角的音量圖示可開啟「重生前 2 分鐘」音效提醒。

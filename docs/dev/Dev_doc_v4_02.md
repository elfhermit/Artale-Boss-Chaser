### ✂️ 第一刀：刪除冗餘資訊 (Must Delete)

2.  **頁面頂部的「Boss 資料檢視 (範例)... 載入自...」文字**

      * **原因**：這是開發者除錯訊息，玩家不需要看這個。
      * **動作**：移除，或是縮小放到頁面最底部的 Footer (版權宣告區)。

3.  **右側輸入框的「頻道 (例如 1)」文字輸入框**

      * **原因**：下面已經有漂亮的「數字鍵盤 (1\~20)」了，這個文字輸入框顯得很多餘。

範例程式碼（可直接複製到 `docs/index.html` 中）：

```html
<div class="channel-control-group">
  <label class="input-label">頻道</label>
  <div class="stepper-wrapper">
    <button type="button" id="channel-dec" class="step-btn">❮</button>
    <input type="number" id="channelInput" value="1" min="1" max="3000" class="channel-input" />
    <button type="button" id="channel-inc" class="step-btn">❯</button>
  </div>
  <div id="quick-channels" class="quick-channels"></div>
</div>

<script>
  function clampChannel(v){ return Math.max(1, Math.min(3000, v)); }
  document.getElementById('channel-inc').addEventListener('click', ()=>{
    const input = document.getElementById('channelInput');
    input.value = clampChannel((parseInt(input.value)||1) + 1);
  });
  document.getElementById('channel-dec').addEventListener('click', ()=>{
    const input = document.getElementById('channelInput');
    input.value = clampChannel((parseInt(input.value)||1) - 1);
  });
  // quick-channels 可由 app.js 填入最近使用紀錄
</script>
```
